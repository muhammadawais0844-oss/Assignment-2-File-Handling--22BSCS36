# Contact Management System (File-Based)
### Assignment 2 - Advanced Programming

## 📌 Description
A Python CLI application that stores contact information in a CSV file. It demonstrates persistent storage and CRUD operations.

## 🛠️ Technical Details
- **Language:** Python 3.x
- **Format:** CSV (Comma Separated Values)
- **Structure:** `data/contacts.csv`

## 🚀 How to Run
1. Go to `source_code/` directory.
2. Run `python contact_manager.py`.

## ✨ Features
- **Persistence:** Data stays saved after closing.
- **Error Handling:** Protects against missing files and I/O errors.
- **Menu-Driven:** Easy navigation via terminal.
